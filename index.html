Projeto Monitoramento de ruído

Sistema interativo que simula níveis de ruído nas cidades.

Saiba mais

Cidades Silenciosas é um projeto desenvolvido com HTML, CSS e PHP, que simula a identificação e análise de níveis de ruído urbano com base na localização geográfica. Embora os valores de decibéis sejam gerados aleatoriamente, o sistema foi planejado de forma lógica e estruturada, representando o comportamento real de uma aplicação conectada a sensores de som. A proposta busca demonstrar, de maneira prática e visual, como a tecnologia pode contribuir para o monitoramento ambiental, unindo programação, sustentabilidade e qualidade de vida nas cidades.

Ao clicar na imagem, você será redirecionado para o repositório do projeto no GitHub, onde estão disponíveis todos os códigos, arquivos e documentação completa do sistema.
